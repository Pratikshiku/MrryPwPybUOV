### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、企业、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 企业
5. 招聘信息
6. 招聘新闻
7. 留言反馈
8. 个人中心
9. 我的收藏



**企业功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 岗位类型管理
6. 招聘信息管理
7. 应聘记录管理







**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 企业管理
4. 用户管理
5. 岗位类型管理
6. 招聘信息管理
7. 应聘记录管理
8. 留言反馈
9. 系统管理（招聘新闻、轮播图管理）
